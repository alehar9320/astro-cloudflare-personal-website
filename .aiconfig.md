# AI Agent Verification Protocol

Before performing a `git push` or finalizing a task, you MUST execute this pipeline. If any step fails, you must fix the code and restart from Step 1.

## 1. Quality Pipeline

- **Format:** `npm run format` (Uses Prettier to ensure style consistency)
- **Lint:** `npm run lint` (Checks for syntax and logic errors via ESLint)
- **Type/Astro Check:** `npm run check` (Validates Astro components and TypeScript)
- **Test:** `npm run test` (Runs Vitest suite to prevent regressions)

## 2. Execution Rules

- **Strict Adherence:** You are prohibited from using `--no-verify` or bypassing these checks.
- **Reporting:** In your task summary, explicitly list which checks were run and confirm they passed.
- **Node Version:** Ensure you are using Node.js >= 22.12.0 as specified in `package.json`.
