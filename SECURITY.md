# Security Policy

## Supported Versions

Only the latest version of this project is supported. Security updates are applied to the `main` branch.

| Version  | Supported          |
| -------- | ------------------ |
| Latest   | :white_check_mark: |
| < Latest | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by contacting [Alexander Härenstam on LinkedIn](https://www.linkedin.com/in/alehar/).
