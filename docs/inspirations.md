# Personal Website Inspirations

A collection of high-quality personal websites and portfolios to draw inspiration from regarding design, structure, and content.

## Notable Examples

- **[Addy Osmani](https://addyosmani.com/)**
  Great example of performance, clear typography, and a focus on writing and projects.
- **[Lee Robinson](https://leerob.io/)**
  Excellent execution of a modern developer portfolio, integrating a blog, guestbook, and metrics dashboard with a clean, minimalistic aesthetic.

- **[Paco Coursey](https://paco.me/)**
  Outstanding micro-interactions, dark mode handling, and a highly polished, minimal design.

- **[Brian Lovin](https://brianlovin.com/)**
  Rich content organization, thoughtful typography, and neat integration of writing and podcasts.

- **[Emil Kowalski](https://emilkowal.ski/)**
  Incredible focus on animations, crafted details, and UI engineering excellence.

- **[Josh W. Comeau](https://www.joshwcomeau.com/)**
  Fantastically interactive blog with custom MDX components and a playful yet readable design.

- **[Rauno Freiberg](https://rauno.me/)**
  Experimental, heavily design-focused, and boundary-pushing web interactions.

- **[Brittany Chiang](https://brittanychiang.com/)**
  A very clean layout often cited as a standard for software engineering portfolios, featuring sophisticated use of layout and scroll interactions.
