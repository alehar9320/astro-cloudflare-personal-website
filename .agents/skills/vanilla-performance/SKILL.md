# Vanilla Performance Optimization

Dedication to building a lean and fast user experience without the use of heavy runtime frameworks.

## When to use this skill

- Writing any new CSS (`.css` or within `.astro`).
- Adding minimal client-side JavaScript.
- Optimizing images and other static assets.
- Configuring the Cloudflare image optimization service.

## How to use it

1. **Vanilla First:** Do NOT introduce utility-first CSS frameworks (like Tailwind) or large JS libraries unless absolutely requested.
2. **Lean Bundling:** Ensure that any client-side JavaScript is as minimal as possible and use dynamic imports for non-critical code.
3. **Edge Optimization:** Use Astro's built-in image service with the `cloudflare-binding` entrypoint for optimized image delivery at the edge.
4. **Visual Style:** Maintain the project's 'Northern Lights' and 'Glassmorphism' principles using high-performance pure Vanilla CSS.
